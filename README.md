# Project Documentation

## Overview
This project provides a complete ESP32 Bluetooth HID keyboard remote control for Android devices, allowing users to control their devices wirelessly.

## Features
- Bluetooth HID functionality
- Control Android devices from a distance
- Customizable pinout configuration
- Easy installation guide

## Hardware Requirements
- ESP32 Microcontroller
- Bluetooth-enabled Android device

## Pinout Configuration
- **Pin 0**: Output for LED
- **Pin 1**: Input for button A
- **Pin 2**: Input for button B

## Installation Guide
1. Install the Arduino IDE.
2. Set up the ESP32 board in Arduino IDE.
3. Upload the provided code to the ESP32.

## Troubleshooting
- Ensure Bluetooth is enabled on the Android device.
- Check the pin connections if the remote isn't working.

## Technical Specifications
- ESP32 chip
- Bluetooth Low Energy support

## Customization Guide
- Modify the pin configuration in the code according to your hardware setup.
